## 🚀 Como publicar no GitHub Pages

```bash
# 1. Clone o repositório
git clone https://github.com/MatheusSSiqueira/portfolio.git
cd portfolio

# 2. Faça suas alterações e commit
git add .
git commit -m "feat: atualiza portfólio"
git push origin main
```

**Ativar o GitHub Pages:**
1. Vá em **Settings → Pages**
2. Source: `Deploy from a branch`
3. Branch: `main` · Folder: `/ (root)`
4. Clique **Save** e aguarde ~1 minuto

> URL gerada: `https://MatheusSSiqueira.github.io/portfolio`

## 🔄 Versionamento — boas práticas

```bash
# Branches recomendadas
main          ← produção (GitHub Pages)
develop       ← desenvolvimento
feature/nome  ← nova funcionalidade

# Commits semânticos
feat:     nova funcionalidade
fix:      correção de bug
style:    ajuste de CSS
docs:     atualização de README
refactor: melhoria de código sem mudança de comportamento
```
