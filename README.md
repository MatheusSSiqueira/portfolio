## ✨ Funcionalidades JavaScript

| # | Feature | Descrição |
|---|---------|-----------|
| 1 | **Typewriter** | Cicla frases animadas no hero com efeito de digitação |
| 2 | **Nav Shadow** | Sombra na barra de navegação ao rolar a página |
| 3 | **Scroll Reveal** | Fade-in com stagger ao entrar no viewport (IntersectionObserver) |
| 4 | **Contador Animado** | Números incrementam com easing cúbico ao entrar no viewport |
| 5 | **Active Nav** | Link da seção atual destacado automaticamente ao rolar |
| 6 | **Back to Top** | Botão flutuante aparece após 400px de scroll |
| 7 | **GitHub API** | Busca contagem real de repositórios em tempo real |

## 🛠 Tecnologias

- **HTML5** — estrutura semântica (`header`, `nav`, `main`, `section`, `article`, `footer`)
- **CSS3** — custom properties, grid, flexbox, sticky nav, animações
- **JavaScript ES6+** — `IntersectionObserver`, `Fetch API`, `requestAnimationFrame`
- **Font Awesome 6** — ícones via CDN
- **Google Fonts** — JetBrains Mono + Inter
